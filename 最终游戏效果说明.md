## 🎮 动画效果 (多帧预览)

生成了 4 帧动画预览，展示动态效果:

| 帧 | 效果 |
|----|------|
| frame_000.png | 初始位置 |
| frame_010.png | 海草右摆，气泡上升，龙虾呼吸 |
| frame_020.png | 海草左摆，气泡继续上升，包裹浮动 |
| frame_030.png | 海草回中，龙虾触角摆动 |

### 动画细节

#### 龙虾呼吸
```python
breath = 1.0 + 0.03 * sin(frame * 0.5)
# 缩放范围：0.97x ~ 1.03x
```

#### 触角摆动
```python
wave = sin(frame * 0.3) * 5
# 左右摆动：±5 像素
```

#### 包裹浮动
```python
float_y = sin(frame * 0.08) * 5
# 上下浮动：±5 像素，较慢节奏
```

#### 海草摇摆
```python
wave = sin(frame * 0.05 + x * 0.1) * 8
# 每丛海草独立摆动，形成波浪效果
```

#### 气泡上升
```python
offset = (frame * 2) % 50
# 持续向上，循环
```
