## 🚀 使用流程

### 1. 在 Unity 中打开项目
```
Unity Hub → Add → 选择 LobsterCourier 文件夹
```

### 2. 生成美术资源
```
菜单栏 → Lobster Courier → 生成美术资源
点击 "生成所有美术资源"
```

### 3. 生成音效资源
```
菜单栏 → Lobster Courier → 生成音效资源
点击 "生成所有音效"
```

### 4. 搭建场景
```
菜单栏 → Lobster Courier → 自动搭建场景
点击 "创建主游戏场景"
```

### 5. 运行游戏
```
保存场景 (Ctrl+S)
点击 ▶️ Play
按 W/A/S/D 移动测试
```
