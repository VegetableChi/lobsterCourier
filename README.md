# 🦞 龙虾快递员 (Lobster Courier)

一款轻松休闲的海底送货游戏

**原创 IP 声明**: 本游戏所有角色、名称、设定均为原创，不引用任何现有版权作品。

---

## 🎮 游戏简介

你是一只海底龙虾快递员，在洋流中穿梭送货，把海鲜外卖送到各个海洋生物手中。

**核心玩法：**
- 操作简单，单手也能玩
- 利用洋流加速，节省体力
- 准时送达赚取小费
- 升级装备，成为海底最快快递员

## 🚀 快速开始

### 前置要求

- Unity 2022.3 LTS 或更高版本
- .NET Framework 4.x

### 安装步骤

1. **打开 Unity Hub**

2. **添加项目**
   - 点击 "Add" → 选择 `LobsterCourier` 文件夹

3. **打开项目**
   - 在 Unity Hub 中点击项目名称

4. **打开场景**
   - 双击 `Assets/Scenes/MainScene.unity`

5. **运行游戏**
   - 点击 Unity 编辑器的 ▶️ Play 按钮

---

## 🎯 游戏操作

| 按键 | 功能 |
|------|------|
| W/A/S/D 或 方向键 | 移动 |
| Shift | 冲刺（消耗体力） |
| E 或 空格 | 抓取/放下包裹 |
| P 或 Esc | 暂停游戏 |
| R | 重新开始（游戏结束时） |

## 🔧 配置说明

### 玩家参数 (LobsterController)

```csharp
moveSpeed = 5f;           // 基础移动速度
sprintMultiplier = 2f;    // 冲刺倍率
maxStamina = 100f;        // 最大体力
staminaDrainRate = 20f;   // 体力消耗速率
staminaRegenRate = 10f;   // 体力恢复速率
```

### 关卡生成 (LevelGenerator)

```csharp
worldSize = 100f;         // 世界大小
obstacleCount = 20;       // 障碍物数量
currentZoneCount = 5;     // 洋流区域数量
deliveryPointCount = 8;   // 送货点数量
```
