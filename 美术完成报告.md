## 📋 使用步骤

### 在 Unity 中使用
1. 运行 SpriteGenerator 工具
2. 等待资源生成完成
3. 在 Project 窗口查看 Sprites 文件夹
4. 将精灵拖拽到对应对象
5. 运行游戏测试

### 配置到对象
```
玩家对象:
- SpriteRenderer → Player_Down_Frame0

NPC 对象:
- SpriteRenderer → Star_Frame0 (或其他 NPC)

背景:
- SpriteRenderer → SeabedBackground

UI:
- Image → Button_Normal / Icon_Coin 等
```
