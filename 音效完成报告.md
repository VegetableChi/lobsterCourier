## 🛠️ 生成工具

### AudioGenerator.cs
**位置:** `Assets/Scripts/Editor/AudioGenerator.cs`

**功能:**
- 程序化合成所有音效
- 使用音频合成技术
- 生成 WAV 格式文件
- 一键生成 12 个音效

**合成技术:**
- 正弦波合成
- 包络控制 (ADSR)
- 和弦合成
- 噪声生成
- 滑音效果

**使用方法:**
```
Unity 菜单栏 → Lobster Courier → 生成音效资源

或选择生成:
• 生成所有音效
• 生成背景音乐 (BGM)
• 生成游戏音效 (SFX)
• 生成 UI 音效
```

## 🔊 在 Unity 中使用

### 配置 AudioManager
```csharp
// 在 AudioManager 中分配音频剪辑
public AudioClip[] bgmClips;
public AudioClip[] sfxClips;

// 分配:
bgmClips[0] = BGM_Menu
bgmClips[1] = BGM_Game

sfxClips[0] = SFX_Move
sfxClips[1] = SFX_Sprint
sfxClips[2] = SFX_Pickup
sfxClips[3] = SFX_Deliver
sfxClips[4] = SFX_Coin
sfxClips[5] = SFX_Achievement
sfxClips[6] = SFX_UI_Click
sfxClips[7] = SFX_UI_Hover
sfxClips[8] = SFX_UI_Purchase
sfxClips[9] = SFX_UI_Error
```

### 播放音效
```csharp
// 播放 BGM
AudioManager.Instance.PlayBGM(0); // 主菜单

// 播放 SFX
AudioManager.Instance.PlaySFX(2); // 拾取音效

// 播放 UI 音效
AudioManager.Instance.PlaySFX(6); // 点击音效
```
