## 🎭 角色设计

### 主角 - 龙虾快递员

**外观特征:**
```
- 身体：橙红色，圆润可爱
- 眼睛：大眼睛，黑色瞳孔，白色高光
- 钳子：一大一小，可爱比例
- 触角：两根长触角，末端小球
- 表情：友好微笑
```

**动画帧:**
```
 idle: 3 帧 (呼吸动画)
 run:  4 帧 (游泳动作)
 grab: 2 帧 (钳子开合)
 happy: 3 帧 (送达庆祝)
```

**颜色方案:**
- 身体：#FF6B4A (橙红)
- 腹部：#FFB6C1 (粉红)
- 眼睛：#FFFFFF (白) + #000000 (黑)
- 钳子：#FF4500 (深橙)

### 8 种 NPC 设计

#### 1. 星星 (海星 - 美食博主)
```
- 形状：五角星形
- 颜色：金黄色 #FFD700
- 特征：戴着小眼镜，5 个角都可以拿东西
- 表情：兴奋、热情
```

#### 2. 八爪 (章鱼 - 艺术家)
```
- 形状：圆形头 + 8 条触手
- 颜色：紫色 #9B59B6
- 特征：戴着贝雷帽，触手上有颜料
- 表情：专注、文艺
```

#### 3. 珊瑚 (珊瑚鱼 - 家庭主妇)
```
- 形状：椭圆身体 + 鱼鳍
- 颜色：彩色条纹 #FF69B4
- 特征：圆润体型，温和表情
- 表情：温柔、友善
```

#### 4. 钳子 (螃蟹 - 商人)
```
- 形状：扁圆身体 + 两只大钳
- 颜色：红色 #DC143C
- 特征：戴着商帽，金色大钳子
- 表情：精明、算账
```

#### 5. 卷卷 (海马 - 图书管理员)
```
- 形状：海马造型，卷曲尾巴
- 颜色：淡蓝色 #87CEEB
- 特征：戴着老花镜，尾巴可以卷东西
- 表情：平静、智慧
```

#### 6. 飘飘 (水母 - 瑜伽教练)
```
- 形状：半透明伞状 + 飘逸触手
- 颜色：淡紫色 #E6E6FA
- 特征：身体会发光，触手飘逸
- 表情：宁静、神秘
```

#### 7. 老海 (海龟 - 退休长老)
```
- 形状：龟壳 + 四足 + 长胡子
- 颜色：绿色 #228B22
- 特征：长胡子，戴着老花镜
- 表情：慈祥、智慧
```

#### 8. 深深 (鲨鱼 - 企业家)

- 形状：流线型鲨鱼身体
