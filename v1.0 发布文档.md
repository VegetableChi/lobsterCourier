## 项目概述

### 游戏信息
| 项目 | 说明 |
|------|------|
| **名称** | 龙虾快递员 (Lobster Courier) |
| **类型** | 休闲送货游戏 |
| **平台** | PC (Windows/Mac/Linux) |
| **引擎** | Unity 2022.3 LTS |
| **语言** | 中文 |
| **版本** | v1.0 |

### 游戏简介
你是一只海底龙虾快递员，在洋流中穿梭送货，把海鲜外卖送到各个海洋生物手中。利用洋流加速，管理体力，准时送达赚取金币，升级你的能力，成为海底最快快递员！

### 特色功能
- 🌊 **洋流系统** - 顺流加速、逆流减速
- 💪 **体力管理** - 冲刺消耗，策略使用
- 🔥 **连击奖励** - 连续送达最高 +200%
- 🏪 **商店升级** - 速度/体力/背包升级
- 🏆 **成就系统** - 12 项成就
- 📅 **每日挑战** - 每日特殊任务
- 🌤️ **天气系统** - 6 种动态天气
- 📖 **教程系统** - 10 步新手引导

## 安装指南

### 系统要求
| 要求 | 最低配置 | 推荐配置 |
|------|----------|----------|
| **操作系统** | Windows 10 | Windows 11 |
| **处理器** | Intel i3 | Intel i5 |
| **内存** | 4 GB RAM | 8 GB RAM |
| **存储** | 500 MB | 1 GB |
| **Unity** | 2022.3 LTS | 2022.3 LTS |

### 安装步骤

#### 方法 1: Unity Hub (推荐)
```
1. 下载并安装 Unity Hub
2. 安装 Unity 2022.3 LTS
3. Unity Hub → Add → 选择 LobsterCourier 文件夹
4. 点击项目打开
5. 等待资源导入完成
```

#### 方法 2: 直接打开
```
1. 打开 Unity 2022.3 LTS
2. File → Open Project
3. 选择 LobsterCourier 文件夹
4. 点击 Open
```

### 首次运行
```
1. 打开项目后
2. 菜单栏 → Lobster Courier → 最终完善
3. 等待 5-10 分钟完成资源生成和场景搭建
4. 打开 GameScene
5. 点击 Play 开始游戏
```

## 快速开始

### 5 分钟开始游戏

#### 步骤 1: 打开项目 (1 分钟)
```
Unity Hub → Add → 选择 LobsterCourier → 打开
```

## 游戏说明

### 核心玩法

#### 移动系统
- **W/A/S/D**: 控制龙虾移动方向
- **Shift**: 冲刺（消耗体力）
- **洋流**: 顺流加速，逆流减速

#### 体力系统
- **体力条**: 绿色表示充足，红色表示不足
- **冲刺**: 消耗体力，速度×2
- **恢复**: 停止冲刺自动恢复

#### 包裹系统

## 技术文档

### 项目结构
```
LobsterCourier/
├── Assets/
│   ├── Scripts/              # C# 脚本 (37 个)
│   │   ├── Core/             # 核心系统
│   │   ├── Editor/           # 编辑器工具 (5 个)
│   │   ├── Extensions/       # 扩展功能
│   │   └── Utilities/        # 工具类
│   ├── Sprites/              # 美术资源 (59 个)
│   ├── Audio/                # 音效资源 (12 个)
│   ├── Scenes/               # Unity 场景
│   └── Prefabs/              # 预制体
├── README.md                 # 项目介绍
├── QUICK_START.md            # 快速启动
├── COMPLETE_GUIDE.md         # 完整指南
└── 其他文档...
```

### 开发工具
| 工具 | 功能 | 位置 |
|------|------|------|
| SpriteGenerator | 生成美术资源 | Editor 菜单 |
| AudioGenerator | 生成音效资源 | Editor 菜单 |
| SceneSetup | 自动搭建场景 | Editor 菜单 |
| TestingTools | 自动化测试 | Editor 菜单 |
| FinalPolish | 最终完善 | Editor 菜单 |

### 技术特性
- **程序化资源生成**: 59 个美术资源 +12 个音效
- **自动化测试**: 7 项自动化测试
- **性能监控**: 实时 FPS/内存监控
- **对象池**: 粒子效果对象池
- **存档系统**: 自动/手动保存
- **输入管理**: 输入缓冲 + 连击检测
